# Simple WebApp which is a project for my study

### run

```sh
$ docker-compose up -d
$ open http://localhost:5555
```

### feature

* Simple CRUD
  * GET /index
  * GET /edit?id=X
  * POST /edit
  * GET /create
  * POST /create
  * DELETE /delete
  * POST /exec
     * Start a function which will be executed by external service.
